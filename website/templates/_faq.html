Help

Have questions? Look below and see if your question is in the frequently asked questions.

Do I need to know code to use JuxtaposeJS?
Absolutely not. JuxtaposeJS is friction-free. If you have the links to two images, you are ready to use JuxtaposeJS.
Can I customize what JuxtaposeJS looks like?
JuxtaposeJS is designed with a minimalist style meant to work with most websites. If you want to customize the appearance, you can change most of how it looks with CSS rules. Read more on the Github Wiki.
